# 3부 급격한 전환점

- 저자의 자전적인 이야기

## 11장 1960년대

- 12세에 어머니가 사준 DIGI-COMP I을 계기로 프로그래밍에 입문
- ECP-18을 통해 처음으로 실제 컴퓨터를 접함
- 16세에 아버지의 도움으로 디지털 이큅먼트 코퍼레이션(DEC) 영업소에서 일하게 됨

## 12장 1970년대

- 처음으로 프로그래머로 고용됨 (A.S.C. Tabulating)
- 학생 기록을 읽어 ID를 부여하고, 갱신된 데이터를 새로운 테이프에 기록하는 작업 수행 (EASYCODER 사용)
- 이후 A.S.C.에서 TAS로 합류
- 비디오 단말기를 활용한 작업 환경 경험

- OMC 근무
- 구조적 프로그래밍 관련 글을 읽고 큰 충격을 받음
- 테라다인 센트럴에 취직

- 소스코드 관리 방식의 변화 경험
  - 각 ROM 칩이 자체적인 벡터 테이블을 가지도록 설계
  - 모든 호출이 해당 테이블을 거치는 간접 호출 구조로 변경
  - 칩 간 의존도를 낮추고, 개별 수정·컴파일·배포 가능
  - 객체 지향 개념의 초기 형태를 체감

- PDP-11 도입

## 13장 1980년대

- C 언어를 처음 접하고 『The C Programming Language』를 읽으며 깊이 빠져듦
- PDP-11 시스템 관리자 역할 수행

- pCCU 개발  
  → 전화번호를 기반으로 해당 위성 중앙 교환국(COLT)에 명령을 전달하는 장치  
  → C와 BOSS 언어를 사용해 구현

- DLU/DRU 시스템 경험
- 벡터 테이블 초기화 구조 도입  
  → 모듈 간 독립적인 개발 및 배포 가능성 확보

- 애플 II 등 개인용 컴퓨터의 등장 경험
- 최초의 디지털 음성 사서함 및 통화 관리 시스템 구축

- 매킨토시 발표 이후 직접 구입
- 개인용 컴퓨터 보급 확산 및 BBS(전자 게시판 서비스) 활성화 경험
- 음성 응답 시스템(VRS) 개발

- Core War 게임 개발
- Smalltalk 언어에 큰 매력을 느낌

- 수리 작업 재정 시스템 구축
- 객체지향에 관심을 가지며, C 기반 객체지향 프레임워크를 직접 만들기로 결심

- 영국으로 이동: 이 시기에는 소프트웨어 기술보다 관리 업무 중심으로 활동
